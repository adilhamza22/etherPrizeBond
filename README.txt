GitHub Repository Description:

Prize Bond System

This repository contains the source code for a decentralized prize bond system implemented on the Ethereum blockchain. The system allows users to issue prize bonds, participate in prize bond draws, and claim prizes. It is built using Solidity for the smart contracts and web3.js for the frontend interaction.

Features:

Issue new prize bonds
Participate in prize bond draws
Claim prizes for winning bonds
View total bonds issued, total prize amount, and remaining prize fund
Technologies Used:

Solidity
Ethereum Blockchain
web3.js
HTML/CSS
Cloning and Running Steps for Cloners:

Prerequisites:

Node.js installed
Ganache or a local Ethereum node running
MetaMask or a similar Ethereum wallet installed
